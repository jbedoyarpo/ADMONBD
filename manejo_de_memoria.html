MANEJO DE MEMORIA

En este módulo, se aborda las diferentes maneras que tiene Oracle para administrar la memoria que se encuentra incluída en la instancia de la base de datos: SGA, PGA. Data Buffer Cache, Shared Pool, Redolog Buffer, entre otros.

Oracle tiene dos maneras de gestionar la memoria de su instancia:

  • Gestión automática: Oracle, automáticamente, va haciendo crecer la memoria, según necesidad.
  • Gestión manual: El DBA debe estar pendiente de que la memoria no se llene.
