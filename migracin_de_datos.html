MIGRACIÓN DE DATOS

Fuente. https://blog.powerdata.es/el-valor-de-la-gestion-de-datos/la-importancia-del-mdm-en-la-migracion-de-datos

En ocasiones necesitamos mover datos de una tabla a otra. Y estos movimientos los necesitamos hacer de acuerdo a unos criterios o condiciones que cumplan los datos. Hay diferentes maneras de hacer estos movimientos.

En este módulo, vamos a explicar algunas maneras que se tienen para migrar datos de una tabla a otra(s).
