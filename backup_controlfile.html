Por otra parte, si queremos hacer un backup de datos y que adicionalmente le haga un bacckup al Control File, podemos usar la siguiente instrucción. Vamos a hacerle un backup a un tablespace llamado PRUEBA, junto con el Control File.

Como se puede observar, hizo el backup del tablespace PRUEBA, es decir, hizo copia a su Data File PRUEBA.DBF. Además, hizo copia del Control File. Observar que el backup hecho por el usuario del Control File se realiza en la carpeta BACKUPSET de la FRA de la base de datos. En cambio, el backup automático que se hace del Control File queda en la carpeta AUTOBACKUP de la FRA de la base de datos.
